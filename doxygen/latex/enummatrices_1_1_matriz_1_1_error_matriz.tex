\doxysubsection{Detailed Description}
Enum para los errores de matriz. Se utiliza para identificar cuando hay errores en los procedimientos a realizar. En ese caso se retorna una matriz que contenga inicializada el campo \char`\"{}error\char`\"{} con alguno de estos errores. Si no tiene ningun error, entonces se pone el que dice NINGUNO.
